## Key Decisions Made

- **State Management:** Riverpod (compile-time safety, async/streaming support)
- **Backend:** PocketBase (SQLite + REST + SSE, no external API layer)
- **Navigation:** GoRouter (native deep linking for `https://popote.io/s/{shareCode}`)
- **Auth:** Device ID (local UUID, zero-account for guests)
- **Real-time:** Riverpod StreamProvider wrapping PocketBase SSE
